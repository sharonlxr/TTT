Context and Summing Variables

Context and summing variables included in the default configuration.

Context Variables

SUBSTATE:  The substate for the tester. It is a subdivision on the E10 states

LOT_ID: The lot being processed

DEVICE_ID: The device for the current lot

TEST_PROGRAM:  The current test program

TEMPERATURE: The target temperature

RETEST: Whether it is a restest ‘Y’ or nor ‘N’

RESCREEN: Whether it is a rescreen ‘Y’ or nor ‘N’

SITES_AVAILABLE: The number of test sites currently available

HANDLER_ID: The handler ID currently in use

LOADBOARD_ID: The load board ID currently in use

OPERATOR_ID: The ID of the operator currently using the tester

Summing Variables

           INSERTIONS: Number of insertions

           DEVICES_PROCESSED: Number of devices tested

           CNT_GOOD: Number of good devices

           TEST_TIME: Test time in milliseconds

           HND_INDEX_TIME: Index time in milliseconds

           SITEX_HB1:  Number of results for site X and bin 1

           SITEX_HB*:  Number of results for site X and any bin number
