Dashboard

Tool Time Tracker includes a dashboard with four different views of the shop floor: Utilization, Production, Maintenance, and Engineering. Tester icons will blink if the time limit allowed for the tester substate is exceeded. Administrator can set up these time limits.

Tool Time Tracker will remember the last view displayed when closing and reopening the user interface.

Step

Action

1

 

Utilization view. Colors reflect the E10 states. Click on Dashboard -> Legend to view color meanings:

2

Production view. Testers turn orange when they have completed 80% of the lot and red after 95%. Click on Dashboard -> Legend to view meaning of each data item.

3

Maintenance view. Testers turn orange if current site performance is under 90% and red if below 50% or in Unscheduled Down state.

 

4

Engineering view. Testers turn orange if last hour yield falls under 80%, red if below 50%:

 

5

Double-click on any tester to display a chart with current and last 24 hours information. Hover mouse over chart to see values for the last 24 hours (state, lot, device, flow, etc). The chart also shows how the site performance and yield (combined and by site) have evolved for the last 24 hours:
