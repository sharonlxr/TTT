Generating a Report

Complete the following steps to generate a report:

Step

Action

1

 

Select the equipment folder or the individual machines you want included in the report by clicking on the equipment folder or a machine icon. Clicking on a folder selects all the machines under that folder. Click again to deselect. You can view the machines sorted by equipment type or location by clicking on the Type or Location icons on the toolbar.

2

Then select the dates for the report. Click on the calendar icon on the toolbar or select Dates for the Dates menu. The following window appears:

Choose raw, shift, day, week or month data at the top. In raw you set the specific start and end date of the report or you can select the shifts, days, weeks or months to include. Use the CTRL key to select multiple entries. CTRL + a will select all the entries

3

Select a report type and grouping from the Type menu:

 

4

For a report in Excel format, click on the Excel Report icon or select Excel Report from the Report menu. An Excel window will pop up. Click on Options. A new window appears.

Then, select Enable the content and click OK.

The report will appear:

 

5

Optionally, click on the HTML Report icon or select HTML Report from the Report menu to generate a report in HTML format. A browser window will pop up with the report.
